PenBridge

Beta v0.6.2

PenBridge is an application that allows you to send pen and touch input from any device to your Windows device. This means, you can use any device, be it a Surface Tablet, an iPad or a Samsung Tablet with a S Pen as a Graphics Tablet for your PC!

Troubleshooting üîß

  • OneNote (App) doesn't recognize pen pressure

    Reinstall OneNote to fix this issue.

  • My eraser does not work in Software x

    Some Software don't recognize the sent pen correctly as flipped and therefore do not enable the eraser correctly. I'm still searching for a solution for that issue.

  • After switching the monitor with "m" the input gets sent nowhere

    Multi-Monitor Support can be flakey, when your primary monitor is not the leftmost monitor with the enumeration "1", since Windows uses a different coordinate system for window positioning and for pointer inputs. You could try setting the monitor you want to send the input to to your primary monitor and then restart PenBridge.
